Installation Instructions

Windows Installation

  1. Download the ProductMatcher-Windows-1.0.0.exe file
  2. Double-click the downloaded file to launch the application
  3. If Windows SmartScreen appears, click "More info" then "Run anyway"
  4. The application will start automatically - no installation needed!
  5. Your data will be stored in %APPDATA%\ProductMatcher\

Note: The first launch may take a few seconds as the app initializes the database.

macOS Installation

  1. Download the ProductMatcher-macOS-1.0.0.dmg file
  2. Open the downloaded DMG file
  3. Drag the Product Matcher app to your Applications folder
  4. Launch Product Matcher from Applications
  5. If macOS blocks the app, go to System Preferences ‚Üí Security & Privacy ‚Üí click "Open Anyway"
  6. Your data will be stored in ~/Library/Application Support/ProductMatcher/

Note: The app is not code-signed yet. We're working on getting an Apple Developer certificate.
