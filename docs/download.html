AMD GPU Compatibility Notice

AMD ROCm support on Windows is limited to newer GPUs (RX 6000/7000/9000 series) and requires Python 3.12 specifically. Older AMD GPUs (RX 5000 series and earlier) are not supported by ROCm and will fall back to CPU mode.

For best GPU experience, we recommend NVIDIA GPUs or Apple Silicon Macs. See our GPU Setup Guide for detailed instructions.

Installation Instructions

Windows Installation

  1. Download the CatalogMatch-Windows-1.0.0.exe file
  2. Double-click the downloaded file to launch the application
  3. If Windows SmartScreen appears, click "More info" then "Run anyway"
  4. The application will start automatically - no installation needed!
  5. Your data will be stored in %APPDATA%\CatalogMatch\
Note: The first launch may take a few seconds as the app initializes the database.

macOS Installation

  1. Download the CatalogMatch-macOS-1.0.0.dmg file
  2. Open the downloaded DMG file
  3. Drag the CatalogMatch app to your Applications folder
  4. Launch CatalogMatch from Applications
  5. If macOS blocks the app, go to System Preferences → Security & Privacy → click "Open Anyway"
  6. Your data will be stored in ~/Library/Application Support/CatalogMatch/
Note: The app is not code-signed yet. We're working on getting an Apple Developer certificate.
